### python - list 기본 문법


- list 생성
- 1. memo = ["a", "b", "c"]
- 2. memo = list(("a","b","c"))
- 데이터 추가 memo.append("abc")
- 데이터 삽입 memo.insert(2, "abc")
- 데이터 삭제
- 삭제 데이터 지정 memo,remove("c")
- 마지막 데이터 삭제 memo.pop()
- 인덱스로 삭제 del memo[3]
- 배열 합치기 memo.extend(리스트 이름)
- 리스트 크기 len(리스트 이름)
