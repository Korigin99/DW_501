### java

- interface(인터페이스)

      * 인터페이스는 추상 클래스이다.
      * 인터페이스는 객체를 가질 수 없다.
      * 인터페이스에는 기본 메서드가 추상메서드이다.
      * 인터페이스에는 인스턴스변수, 인스턴스메서드를 정의할 수 없다.
      * 인터페이스에는 추상메서드와, static, final만 사용할 수 있다.
      * 인터페이스는 다중 상속이 가능하다.
      * 인터페이스를 이용하여 다형성 구현이 가능하다.
